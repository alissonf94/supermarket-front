# Market React App

This is a React web application project for a virtual supermarket. The app allows users to browse products, add items to the shopping cart, and make online purchases.

## Prerequisites

Before you begin, make sure you have installed:

- [Node.js](https://nodejs.org/)
- [npm](https://www.npmjs.com/) or [Yarn](https://yarnpkg.com/)

## Setup

1. Clone this repository to your local system:

   ```bash
   git clone https://github.com/alissonf94/supermarket-front.git
   ```

2. Navigate to the project directory:

   ```bash
   cd supermarket-front
   ```

3. Install project dependencies:

   ```bash
   npm install
   # or
   yarn install
   ```

## Usage

After completing the setup, you can start the application:

```bash
npm start
# or
yarn start
```

This will start the application in development mode. Open [http://localhost:3000](http://localhost:3000) in your browser to view the app.

## Features

- After logging in, you can enter with two modes: employee or user.
- Depending on the login mode, you will have specific routes for that type of login.
- To log out, simply click the logout button.

**User**

- View the list of available products.
- View the list of products on sale.
- Add products to the shopping cart.
- Remove items from the cart.
- Update user information.
- View the user's purchase history.

**Employee**

- View a table of available products.
- View a table of products on sale.
- Add a product.
- Add a promotion.
- Remove an item.
- Remove a promotion.
